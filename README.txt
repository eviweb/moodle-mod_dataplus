This is a Moodle module enabling the creation of discreet SQLite3 databases in Moodle 2.0.

It was created by Anthony Forth of The Open University (a.j.forth@open.ac.uk).

To install, unzip the dataplus directory to your Moodle mods directory.  Then go to Moodle
administration to update the Moodle database.

Further information on this module can be found at http://www.open.ac.uk/blogs/dataplus
